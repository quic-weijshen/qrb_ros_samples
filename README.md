# Experimental Agentic Framework Demo for Qualcomm Edge AI

## Overview

This repository provides a **Python-based demonstration** of an **agentic framework** designed to showcase **edge AI capabilities on Qualcomm platforms**. It serves as a practical example of how agent-driven architectures can integrate with modern AI workflows at the edge.

***

## Inspiration

This project draws inspiration from the following outstanding open-source efforts:

*   <https://github.com/humanlayer/12-factor-agents>
*   <https://github.com/The-Pocket/PocketFlow>

We deeply appreciate the authors and the open-source community for their creativity and collaborative spirit.

## A Few Words

Edge AI combined with agentic frameworks unlocks new possibilities for **autonomous decision-making**, **context-aware processing**, and **scalable intelligence** at the device level. This repository aims to make these concepts tangible and actionable for developers and researchers.

***

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/qualcomm-qrb-ros/qrb_ros_samples.git --branch experimental
    ```
2.  Install dependencies:
    ```bash
    cd <your-chosen-cookbook>
    pip install -r requirements.txt
    ```
3.  Run the demo:
    ```bash
    python <demo-python-code>
    ```
